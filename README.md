# Quantitative Trading Strategy Analysis

This repository documents my self-taught journey into quantitative finance. The goal of this project is to apply my engineering and data analysis background to the financial markets by programmatically developing, backtesting, and analyzing trading strategies.

## üöÄ Project Objective

The primary objective is to build a personal framework in Python to:
1.  **Automate Market Analysis:** Move beyond manual technical/fundamental analysis (as practiced on TradingView/Investing.com) to programmatic analysis.
2.  **Backtest Strategies:** Quantify the performance of trading ideas (e.g., moving average crossovers, momentum) against historical data.
3.  **Risk Management:** Implement basic risk management models and analyze strategy performance metrics like Sharpe Ratio, Drawdown, and Alpha.
